## 🔐 安全最佳实践

### 1. API Key 管理
- **不要硬编码**: 不要在源代码中直接写入API Key
- **本地存储**: 使用uni-app的本地存储机制
- **定期轮换**: 建议每3-6个月更换一次API Key

### 2. 使用监控
- **关注使用量**: 定期查看DeepSeek平台的使用统计
- **设置预算提醒**: 在平台设置每月使用限额和提醒
- **异常检测**: 如发现异常调用量，立即撤销Key

### 3. 生产环境建议
- **后端中转**: 生产环境建议使用后端服务中转API请求
- **环境变量**: 使用环境变量管理敏感配置
- **访问限制**: 配置IP白名单或访问频率限制

### 4. 数据安全
- **用户数据**: 不要将用户敏感信息发送给AI API
- **日志记录**: 避免在日志中记录完整API Key
- **传输加密**: 确保所有API请求使用HTTPS
