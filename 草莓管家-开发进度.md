## 📝 已知问题

### 功能性问题
1. **H5 环境跨域限制**
   - 现象：某些环境下直接请求 API 可能被浏览器拦截
   - 临时方案：开发时配置代理，生产环境需后端中转
   - 根本解决：部署后端服务

2. **API Key 安全风险**
   - 现象：API Key 存储在客户端 localStorage
   - 风险：可能被恶意提取，导致费用损失
   - 解决方案：后端服务中转 API 请求

3. **语音播报兼容性**
   - 现象：某些浏览器不支持 Web Speech API
   - 影响：H5 环境部分用户无法使用语音功能
   - 降级方案：显示不支持提示，提供文字替代

4. **存储空间限制**
   - 现象：本地存储仅保留最近 50 条对话
   - 影响：历史记录有限
   - 解决方案：后端服务 + 数据库存储

### 技术性问题
1. **SSE 连接稳定性**
   - 现象：弱网环境下流式响应可能中断
   - 当前：基础错误处理
   - 改进：连接保持、自动重连

2. **内存管理**
   - 现象：长时间使用可能内存增长
   - 当前：50条消息限制
   - 监控：需观察实际内存使用情况

   - 现象：小程序、App、H5 部分 API 行为不同
   - 当前：使用条件编译适配
   - 测试：需各平台充分测试

## 🚨 风险与应对

   - 应对：服务降级，备用方案，监控告警

   - 风险：API Key 泄露，数据安全
   - 应对：后端服务，输入验证，安全审计

   - 风险：某些功能在特定平台不可用
   - 应对：充分测试，条件编译，功能降级

   - 应对：用户调研，快速迭代，收集反馈

   - 风险：同类产品竞争
   - 应对：差异化功能，专注草莓种植细分领域
